The honest story — things that went wrong 💀
🔥
Firebase disaster. Spent a whole session integrating Google Sign-In. Auth worked locally, but GitHub Pages kept throwing "unauthorized domain" errors. Removed everything. Gone.
👻
Service Worker nightmare. The app cached itself so aggressively that updates wouldn't show. Stuck on broken old versions for hours. Took 3 separate attempts to fix the caching logic.
⬛
Black screen of death. After removing Firebase, the whole app went blank. JS was running before the DOM loaded. DOMContentLoaded saved the day — eventually.
🗃️
Incomplete file uploads. Multiple times, an uploaded index.html was only 2099 lines — exactly half the file. The other half just vanished. Still don't know why.
📋
Changelog rendering bug. v1.4.0 was missing a closing </div> tag in the modal. The layout broke silently for an entire version with no console errors.
🎨
Scope creep × 10. "Let me add a dark mode" became 39 themes, a custom theme builder, 15 fonts, and a color wheel picker. Still adding more.
💥
Syntax error cascade. Adding CSS inside a JS template literal broke the entire script silently. Two separate crashes from the same pattern in one session.
🤖
Built with Claude (AI). Every bug, fix, and feature in this app was built collaboratively with Claude by Anthropic — all in one chat. 100% transparent about it.
FAQ
Is lucode free?+
Yes, completely free forever. No ads, no subscriptions, no paywalls, no tracking. Just a typing app built for fun by a 14 year old.
Do I need to create an account?+
No. You can use "continue as guest" and everything works. An account lets you keep your stats and settings across devices — but it's optional.
Is my data safe? Does it go to a server?+
Nothing ever leaves your device. Everything — accounts, stats, settings, themes — is stored in your browser's localStorage. No server, no database, no cloud.
What's the difference between lucode and MonkeyType?+
MonkeyType uses random words. Lucode uses real code snippets across 9 languages — so you're actually practicing typing the stuff you'd write as a developer. There's also a run code panel, syntax highlighting, and a typing race vs AI.
Can I use my own code to practice?+
Yes! Click "paste your code" in the header or hit Ctrl+V. Paste any code and type it. Great for memorizing your own functions or algorithms.
What programming languages are supported?+
HTML, CSS, JavaScript, TypeScript, Python, Rust, Lua, Go, and C++. Syntax highlighting and real snippets for all of them.
Can I run the code I type?+
Yes! After finishing, click "▶ run code". HTML/CSS/JS runs live in the panel. Python, Rust, Go, Lua, C++ open in OneCompiler. You can also drag the panel to resize it or go fullscreen.
Why does the app sometimes show a blank screen?+
It's a service worker cache issue. Fix: open DevTools (F12) → Application tab → Storage → Clear site data → hard reload with Ctrl+Shift+R. Or open in incognito mode.
How does the streak work?+
Complete at least one typing session per day. Your streak counter (🔥) in the header goes up by 1 each day you play. Miss a day and it resets to 0. Your best streak is saved forever.
What are XP and levels?+
Every session gives you XP based on your WPM × accuracy. Levels go from Novice all the way to 10x Dev. Click the 🔥 streak button in the header to see your progress and all achievements.
Can I download lucode to use offline?+
Yes! Go to Settings → Account → "Download as offline app". It downloads lucode.html — a single file that works in any browser with no internet needed.
Why only 14 years old?+
Started coding at 12. Built first real projects at 13. lucode is the biggest thing so far — 5000+ lines of code built solo with AI assistance. Age is just a number. Shipping is what counts.
